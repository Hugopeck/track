## Context
When Track's GitHub link is shared on X, Discord, Slack, etc., the social preview card is the first visual impression. It needs to communicate the value prop instantly.

## Design brief
- Text: "Track — Project management tools are dead. A folder is all you need."
- Show a minimal `.track/` folder tree or terminal screenshot
- Dark background, clean typography
- 1280x640px (GitHub's recommended OG image size)
- Set via GitHub repo settings → Social preview

## Tools
- Figma, Canva, or even a well-styled HTML screenshot
- Can also use `og-image` generators

## Acceptance Criteria
- [ ] Social card designed (1280x640)
- [ ] Uploaded as repo social preview in GitHub settings
- [ ] Verified it renders correctly when sharing GitHub link on X
