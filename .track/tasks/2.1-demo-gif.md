## Context
The README needs a visual hook — a 30-second silent GIF showing the Track workflow end-to-end. This is the single most important conversion element for GitHub visitors.

## What to capture
1. `claude /track:init` in a fresh repo
2. `/track:create Add user authentication` — task file appears
3. `/track:work` — agent picks task, opens draft PR
4. Show TODO.md updating with live status
5. PR merged → task auto-completed

## Recording instructions
- Use a tool like `gifox`, `licecap`, or `asciinema` + `svg-term`
- Terminal font: 14-16pt, dark theme
- Resolution: 800x500 or similar (GitHub README width)
- Keep it under 30 seconds, no audio
- Add to README.md where the `<!-- TODO: Add demo GIF here -->` comment is

## Acceptance Criteria
- [ ] GIF recorded showing full init → work → complete cycle
- [ ] GIF embedded in README.md
- [ ] File size under 5MB (optimize with gifsicle if needed)
- [ ] Looks good on both light and dark GitHub themes
