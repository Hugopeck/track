## Context
The launch thread (tweet 6), DEV.to article, and Reddit posts all reference a Track vs Linear comparison table. This needs to exist as a shareable image — not just markdown text.

## Comparison points

| | Linear | Track |
|---|---|---|
| Cost | $8-16/seat/month | Free forever (core) |
| Infra | Cloud SaaS, vendor lock-in | Git. That's it. |
| Agent support | Linear Agent (proprietary) | Any agent (Claude, Cursor, Codex, Gemini) |
| Philosophy | "We're the system" | "The filesystem is the system" |
| Data ownership | Their servers | Your repo |
| Non-code projects | No | Yes — any folder, any project |

## How to create
- Use Figma, Canva, or a screenshot of a well-styled HTML table
- Dark background preferred (matches developer audience)
- Include Track logo/wordmark if available
- Save as PNG, optimize for X (1200x675 or 16:9)
- Save to `growth/assets/linear-comparison.png`
