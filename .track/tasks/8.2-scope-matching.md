## Context
Add `track_match_files_to_task()` function to track-common.sh. This is the deterministic pre-LLM filter used by the post-commit hook to match changed files to tasks.

Input: list of changed file paths (from `git diff --name-only`).
Output: matched task ID(s) + confidence (`deterministic` / `ambiguous` / `unmatched`).

Reuse the existing glob approach from `track_globs_overlap_serialized` in track-common.sh. The function iterates all task files' `files:` globs and checks which ones match the changed files.

- Single match → `deterministic`
- Multiple matches → `ambiguous` (server's LLM resolves)
- No match → `unmatched` (logged as untracked activity)

Must be bash 3.2+ compatible.
