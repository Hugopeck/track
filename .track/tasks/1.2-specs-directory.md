## Context
Track needs a durable location for architecture specs, design docs, and interface contracts that live alongside tasks and plans but don't expire. Plans auto-expire after 7 days — specs are permanent.

Create `.track/specs/` as a new first-class directory in the Track layout. It should be:
- Committed to git (not gitignored)
- Deployed by init to adopting repos
- Documented with a README explaining conventions

Update init's install-manifest.json and SKILL.md to deploy the specs directory and its README.
