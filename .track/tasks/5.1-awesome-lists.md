## Context
The awesome-claude-code lists are primary discovery channels for Claude Code plugins. Getting listed early means being found first. There are 5+ active lists.

## Target lists

1. **hesreallyhim/awesome-claude-code** — "A curated list of awesome skills, hooks, slash-commands, agent orchestrators, applications, and plugins"
   - Read CONTRIBUTING.md before submitting
   - Category: Plugins or Agent Orchestrators

2. **ccplugins/awesome-claude-code-plugins** — "Awesome Claude Code plugins — a curated list of slash commands, subagents, MCP servers, and hooks"
   - Category: Task Management / Coordination

3. **ComposioHQ/awesome-claude-plugins** — "A curated list of Plugins that let you extend Claude Code"
   - Likely has contribution guidelines

4. **rohitg00/awesome-claude-code-toolkit** — "The most comprehensive toolkit for Claude Code"
   - Massive list, high visibility

5. **jqueryscript/awesome-claude-code** — "A curated list of awesome tools, IDE integrations, frameworks"

## Steps per list
1. Fork the repo
2. Read CONTRIBUTING.md and CODE_OF_CONDUCT.md
3. Add Track entry with description matching the list's style
4. Submit PR with clear description
5. Follow up if no response in 3-5 days

## Acceptance Criteria
- [ ] PRs submitted to at least 3 awesome lists
- [ ] Each PR follows the list's contribution guidelines
- [ ] Track description is concise and accurate
- [ ] At least 2 PRs merged

## Notes
Don't submit to all on the same day — stagger to avoid looking spammy.
