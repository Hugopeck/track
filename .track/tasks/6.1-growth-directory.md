## Context
The `growth/` directory is the operational hub for all marketing content. Templates, drafts, checklists, and tracking docs — organized so content creation is fast and repeatable.

## Directory structure
```
growth/
  README.md              — Overview and how to use this directory
  posts/                 — Social media posts (X, Reddit, Discord, HN)
    x-launch-thread.md
    hn-show-hn.md
    reddit-*.md
    discord-intro.md
    weekly-threads/      — Batch-drafted weekly X threads
    outreach/            — Partnership pitches
  articles/              — Long-form content (DEV.to, Medium, blog)
    devto-launch.md
    templates/
      article-template.md
  videos/                — Video scripts and recording checklists
    demo-5min-script.md
    60sec-script.md
    templates/
      video-checklist.md
  tracking/              — Metrics and submission tracking
    content-calendar.md  — 12-week publishing schedule
    metrics-weekly.md    — Weekly metrics template
    submissions.md       — Platform submission tracker
```
