# Growth Content

## Goal
Create the content engine that drives Track from 0 to 1,000 stars. X threads, long-form articles, demo videos, and community posts — each independently valuable, collectively building the "project management is dead" narrative.

## Why Now
Launch window is open. Linear Next validated the thesis. Content created this week has maximum relevance.

## In Scope
- Launch thread (8-10 tweets) for X
- Weekly X threads (1-2/week for 12 weeks)
- Long-form articles (2/month) for DEV.to, Medium, HN
- Demo videos (2-3/month) for YouTube + X
- Reddit posts for r/ClaudeAI, r/programming, r/opensource
- HN "Show HN" submission
- DEV.to launch article
- Product Hunt launch
- All content assets stored in `growth/` directory

## Out Of Scope
- Paid promotion or ads
- Influencer partnerships (that's project 5)
- Content about Track Pro features (premature)

## Shared Context
See `.track/plans/launch-1000-users.md` for full content calendar and thread drafts. The `growth/` directory at repo root holds all content assets, drafts, and scripts.

## Dependency Notes
README must be launch-ready (project 2) before posting. The vendor-neutral install and support story from project 3 strengthens content, but it is not blocking.

## Success Definition
Week 1: Launch thread posted, HN submitted, 3+ Reddit posts live. Week 4: 4+ threads posted, 1 article published, 1 video live. Week 12: Consistent cadence, "project management is dead" associated with Track.

## Candidate Task Seeds
- Write and finalize launch thread
- Draft HN "Show HN" post
- Write DEV.to launch article
- Draft Reddit posts (per-subreddit)
- Script 60-second demo video
- Script 5-minute deep-dive video
- Create video recording instructions
- Draft weekly thread ideas (batch of 8)
- Product Hunt listing prep
