# Growth

Operational hub for Track's marketing and community growth. Everything needed to create content, track submissions, and maintain a consistent publishing cadence.

## Structure

```
posts/              Social media (X threads, Reddit, Discord, HN)
  weekly-threads/   Batch-drafted weekly X threads
  outreach/         Partnership and collaboration pitches
articles/           Long-form content (DEV.to, Medium, blog)
  templates/        Article outline templates
videos/             Video scripts and recording checklists
  templates/        Video checklist templates
assets/             Images, GIFs, social cards, screenshots
tracking/           Metrics, calendar, submission tracker
```

## Workflow

1. Pick a content task from `.track/tasks/` (project 4 or 6)
2. Open the relevant template in this directory
3. Draft the content
4. Publish and update the submission tracker
5. Log metrics weekly

## Key files

- `tracking/content-calendar.md` — 12-week publishing schedule
- `tracking/metrics-weekly.md` — Weekly metrics template
- `tracking/submissions.md` — Where we've posted and the status
