# Assets

Visual assets for growth content. Referenced by posts, articles, and README.

## Naming convention
- `demo-gif.gif` — 30-second README demo
- `linear-comparison.png` — Track vs Linear table
- `social-card.png` — Open Graph image for GitHub/X
- `screenshot-*.png` — Screenshots of Track in action (e.g., `screenshot-todo.png`)

## Specs
- **X images:** 1200x675 (16:9), PNG, under 1MB
- **GIFs:** 800x500, under 5MB, optimize with gifsicle
- **Social card:** 1280x640 (GitHub OG spec)
