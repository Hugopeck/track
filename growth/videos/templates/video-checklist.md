# Video Recording Checklist

**Video title:** [Title]
**Length target:** [Duration]
**Platform:** [YouTube / X / Shorts]

## Pre-Recording
- [ ] Script/outline finalized
- [ ] Demo repo prepared with realistic content
- [ ] Track installed and working in demo repo
- [ ] All tools open (Conductor, terminal, GitHub, etc.)
- [ ] Screen recording software ready (OBS / QuickTime / Loom)
- [ ] Microphone tested (if voiceover)
- [ ] Terminal settings: 16pt font, dark theme, clean prompt
- [ ] Resolution: 1920x1080 (or 1080x1920 for Shorts)
- [ ] Notifications silenced on all devices
- [ ] Practice run completed (check timing)

## Recording
- [ ] Record in segments if easier than one take
- [ ] Show real commands, real output (no faking)
- [ ] Pause between major sections (easier to edit)
- [ ] If voiceover: speak clearly, moderate pace

## Post-Production
- [ ] Edit: cut dead air, loading screens, mistakes
- [ ] Add text overlays for key points
- [ ] Add intro/outro cards
- [ ] Create thumbnail (1280x720)
- [ ] Export: MP4, H.264

## Publishing
- [ ] Upload to YouTube
  - [ ] SEO title (include "AI agents", "project management", relevant keywords)
  - [ ] Description with links (GitHub, install command)
  - [ ] Tags
  - [ ] Thumbnail uploaded
  - [ ] End screen/cards added
- [ ] Share on X with pull quote
- [ ] Embed in README if applicable
- [ ] Add to relevant playlists
