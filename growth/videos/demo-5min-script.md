# Demo Video: "Linear just proved project management is dead. Here's what replaces it."

**Length:** 5-7 minutes
**Platform:** YouTube (primary), X (clip), LinkedIn
**Style:** Screen recording with voiceover (or text overlays if no voiceover)

---

## Script

### Scene 1: Hook (0:00-0:30)
**Voiceover:** "Linear just announced AI agents and admitted issue tracking was built for handoffs, not execution. But their answer is still 'use our SaaS tool.' Mine is a folder."

**Screen:** Show Linear Next announcement page, then transition to a terminal.

### Scene 2: The Problem (0:30-1:30)
**Voiceover:** "Watch what happens when I run three AI agents on the same repo with no coordination."

**Screen:**
- Open Conductor with 3 workspaces
- Each agent starts coding
- Show two agents opening the same file
- Show conflicting PRs on GitHub
- "This costs real time and money."

### Scene 3: Enter Track (1:30-2:00)
**Voiceover:** "Track solves this with a folder. Let me show you."

**Screen:**
- Fresh terminal in a real repo
- Type: `claude` to open Claude Code
- Type: `/track:setup-track`
- Show the `.track/` folder being created
- Show CLAUDE.md being updated

### Scene 4: Create Tasks (2:00-3:00)
**Voiceover:** "Now let's break a real goal into tasks."

**Screen:**
- Type: `/track:decompose Migrate the API to v2`
- Show Track reading the codebase
- Show proposed tasks with file scopes
- Confirm creation
- Show `TODO.md` as the execution queue and `BOARD.md` as the project map

### Scene 5: Parallel Agents (3:00-5:00)
**Voiceover:** "Now the magic. Three agents, three tasks, zero conflicts."

**Screen:**
- Open Conductor with 3 workspaces
- Each runs `/track:work`
- Show each agent picking a different task
- Show file scopes preventing overlap
- Show draft PRs appearing on GitHub
- Show `TODO.md` and `BOARD.md` updating with live status
- Merge one PR
- Show task auto-completing

### Scene 6: The Vision (5:00-5:30)
**Voiceover:** "No account. No subscription. No server. Just git. And if a folder can manage code, why not anything?"

**Screen:** Show `.track/` folder examples for non-code projects

### Scene 7: CTA (5:30-6:00)
**Voiceover:** "Track is free and open source. Star it on GitHub, install it in one command, and follow me for updates."

**Screen:** Show GitHub page, star button, install command
